# carHUD
Amazing heads up display for your car!

# Features
- Track and display MPH
- Track and display RPM

# How to run
In a windows command line:
1. cd to carHUD directory
2. pip install -r requirements.txt
3. py test.py

In a linux command line:
1. cd to carHUD directory
2. pip install -r requirements.txt
3. python test.py
